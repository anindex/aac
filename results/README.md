# Experiment Results Index

This directory holds every CSV/JSON/log that backs a Table or Figure in the
paper, plus a small number of audit artifacts referenced from the prose. The
codebase has two canonical entry points:

```bash
python scripts/reproduce_paper.py                 # full pipeline (~hours): runs every experiment
python scripts/reproduce_paper.py --tables-only   # plot/table only (seconds): preflights CSVs, regenerates outputs
python scripts/reproduce_paper.py --track dimacs  # one track (see --help for the 11 valid tracks)
```

`--tables-only` is the **single source of truth for plotting and table
regeneration**: it preflights every required CSV/JSON, fails fast with an
actionable per-track diagnostic if anything is missing, then runs all figure
scripts and the paper-table consistency drift detector
([`scripts/check_paper_consistency.py`](../scripts/check_paper_consistency.py)).
The full pipeline additionally re-runs every heavy experiment to regenerate
the inputs.

## Pre-registration Record

Before running any evaluation on the OGB-arXiv citation graph (Sec.5.9.3 of the paper), the following prediction was filed with a timestamped commit. It is reproduced here verbatim so that the audit trail lives alongside the artifact rather than only inside the paper's appendix:

> on OGB-arXiv (citation graph, ~170K nodes, non-metric, high clustering), AAC should beat ALT at every budget level with a gap of roughly +2 to +6 percentage points at 32 B/v narrowing to +1 to +3 pp at 128 B/v; the hybrid max(h_AAC, h_ALT) should be the top performer; admissibility violations should be zero.

The prediction was based on the covering-radius theory (Theorem 2 / Corollary 1 in the paper) and the synthetic-graph findings (Sec.5.9). The evaluation outcome -- including the directionality-aware accounting correction that subsequently invalidated the magnitude bands and the per-budget falsification analysis -- is reported in the paper's Appendix E.

The full pre-registered protocol, predicted vs. observed outcome, and the accounting convention are documented in [`PREREG.md`](PREREG.md).

## Provenance and drift check

Every `table_*.tex` file carries a machine-readable `%%% PAPER-TABLE-PROVENANCE`
header that names its generator (a script path, or `hand-maintained`) and the
exact CSV(s) it draws from. The verifier
[`scripts/check_paper_consistency.py`](../scripts/check_paper_consistency.py)
parses every header, tokenizes every numeric cell of the corresponding table,
and confirms each value can be located in the named source CSVs (with
rounding tolerance and LaTeX scientific-notation conversion). It is wired into
`scripts/reproduce_paper.py --tables-only` as the final step.

```bash
python scripts/check_paper_consistency.py                  # all 20 tables
python scripts/check_paper_consistency.py table_X.tex  # one table
python scripts/check_paper_consistency.py --strict         # WARN -> exit 1
```

WARN entries are typical for **derived** values (per-seed means, Fisher /
Stouffer combinations, gap deltas) that are computed from the CSVs but do
not appear literally in any single row; the verifier prints sample contexts
so a human reviewer can confirm them. FAIL means a header is missing or a
source CSV is gone.

## Reproduction

```bash
# Full pipeline (all experiments + tables + figures + verification, ~hours)
python scripts/reproduce_paper.py

# Plot/table only (seconds): preflights every required CSV, runs all figure
# scripts and the consistency drift check; fails fast if a CSV is missing.
python scripts/reproduce_paper.py --tables-only

# Single track (one of the 11 tracks; see --help)
python scripts/reproduce_paper.py --track dimacs
python scripts/reproduce_paper.py --track osmnx
python scripts/reproduce_paper.py --track hybrid
python scripts/reproduce_paper.py --track synthetic
python scripts/reproduce_paper.py --track ablation
python scripts/reproduce_paper.py --track training_drift
python scripts/reproduce_paper.py --track cdh
python scripts/reproduce_paper.py --track coverage
python scripts/reproduce_paper.py --track contextual
python scripts/reproduce_paper.py --track query
python scripts/reproduce_paper.py --track figures
```

Run logs (`*_log.txt`, `*.log`) are gitignored and not tracked.
